## 辅助多项式

对于一些需要用泰勒中值定理解决的题,其实可以采用**辅助多项式**的方法搞定。所谓辅助多项式,指的是当欲证结论为“$f^{(n)}(\xi) = k$, 且 $k \neq 0$”之类的形式, 且题干中关于 $f(x)$ 的信息较多时, 我们可以构造一个 $n$ 次多项式 $P(x)$, 使得 $P(x)$ 满足题干中 $f(x)$ 所满足的一切条件, 再令 $F(x) = f(x) - P(x)$

此时 $F(x)$ 一定会有很多的零点, 然后对 $F(x)$ **多次使用罗尔定理**即可得到欲证结论。

**示例1**设 $f(x)$ 在 $[0,1]$ 二阶可导, $f(0) = 0, f(1) = 1, f(4) = 2$, 证明: $\exists \xi \in (0,4)$, 使得 $f''(\xi) = -\dfrac{1}{3}$

!!! Proof
    > 我们想证明 $f''(\xi) = -1/3$。构造一个二次多项式 $P(x)$，使得 $P''(x) = -1/3$。
    >
    >$P''(x) = -1/3 \implies P(x) = -x^2/6 + C_1x + C_2$。
    >
    >我们希望 $P(x)$ 能拟合 $f(x)$ 的已知条件，令 $P(0)=f(0), P(1)=f(1), P(4)=f(4)$。
    >
    > 用待定系数法可知 $P(x) = -\dfrac{x^2}{6} + \dfrac{7x}{6}$。

    ---

    令辅助函数 $h(x) = f(x) - P(x) = f(x) - (-\dfrac{x^2}{6} + \dfrac{7x}{6}) = f(x) + \dfrac{x^2-7x}{6}$

    则 $h(0) = h(1) = h(4) = 0$

    对 $h(x)$ 在区间 $[0,1]$ 和 $[1,4]$ 上分别应用罗尔定理，可知：
    
    存在 $\xi_1 \in (0,1)$ 和 $\xi_2 \in (1,4)$ 使得 $h'(\xi_1)=0$ 和 $h'(\xi_2)=0$。
    
    再对 $h'(x)$ 在区间 $[\xi_1, \xi_2]$ 上应用罗尔定理，可知存在 $\xi \in (\xi_1, \xi_2) \subset (0,4)$ 使得 $h''(\xi)=0$

    而 $h''(x) = f''(x) - P''(x) = f''(x) - (-1/3) = f''(x) + 1/3$。
    所以 $h''(\xi) = f''(\xi) + 1/3 = 0$，即 $f''(\xi) = -1/3$

**示例3** 设 $f(x)$ 在 $[0,1]$ 二阶可导, $f(0) = f(1) = 0, [f(x)]_{\min} = -1$, 证: $\exists \xi \in (0,1)$, s.t. $f''(\xi) \ge 8$

!!! Proof

    令$P(x) = 4(x^2-x)$

    令 \( F(x) = f(x) - P(x) = f(x) - 4x^2 + 4x \)，则 \( F(0) = F(1) = 0 \)

    假设 \( f(x) \) 在 \( x = c \) 处取最小值，即 \( f(c) = -1 \)，而 \( P\left(\frac{1}{2}\right) = -1 \)。

    1. **若 \( c = \frac{1}{2} \)**
    则 \( F(c) = F\left(\frac{1}{2}\right) = f\left(\frac{1}{2}\right) - P\left(\frac{1}{2}\right) = (-1) - (-1) = 0 \)，故 \( F(0) = F(c) = F(1) = 0 \)。
    由**反复罗尔定理**，存在 \( \xi \in (0,1) \)，使得 \( F''(\xi) = 0 \)，即 \( f''(\xi) = 8 \geq 8 \)，得证。

    2. **若 \( c \neq \frac{1}{2} \)**
    则 \( F(c) = f(c) - P(c) < 0 \)；\( F\left(\frac{1}{2}\right) = f\left(\frac{1}{2}\right) - P\left(\frac{1}{2}\right) > 0 \)。
    由**零点定理**，存在 \( x_0 \in (0,1) \)，使得 \( F(x_0) = 0 \)，故 \( F(0) = F(x_0) = F(1) = 0 \)。
    由**反复罗尔定理**，存在 \( \xi \in (0,1) \)，使得 \( F''(\xi) = 0 \)，即 \( f''(\xi) = 8 \geq 8 \)，得证。

    特别注意的是，本题中 \( f(x) \) 和 \( P(x) \) 虽然拥有相同的最小值 \(-1\)，但它们取到最小值的横坐标却不一定相同，所以需要分类讨论。

    **类题(2007考研)** 设 $f(x), g(x)$ 在 $[a,b]$ 上连续, 在 $(a,b)$ 内二阶可导, $f(a) = g(a), f(b) = g(b)$, 并且 $f(x)$ 和 $g(x)$ 在 $(a,b)$ 内存在相等的最大值, 证明: 存在 $\xi \in (a,b)$, 使得 $f''(\xi) = g''(\xi)$

**示例4** 设 $f(x)$ 在 $[-1,1]$ 三阶连续可导, $f(-1) = 0, f'(0) = 0, f(1) = 1$, 证: $\exists \xi \in (-1,1)$, s.t. $f'''(\xi) = 3$

> 具体情况具体分析，缺少条件的时候，可以人为加一个条件$F(0) = 0$去构造

!!! Proof
    令 $F(x) = f(x) - \left[ \dfrac{1}{2}x^3 + (\dfrac{1}{2}-f(0))x^2 + f(0) \right]$。
    根据我们的构造：
    $F(-1) = 0, F(1) = 0, F(0) = 0, F'(0) = 0$。

    因为 $F(-1)=F(0)=F(1)=0$，对 $F(x)$ 在 $[-1,0]$ 和 $[0,1]$ 上应用罗尔定理：
    存在 $\theta_1 \in (-1,0)$ 使得 $F'(\theta_1)=0$。
    存在 $\theta_2 \in (0,1)$ 使得 $F'(\theta_2)=0$。

    现在我们有 $F'(\theta_1)=F'(0)=F'(\theta_2)=0$。
    对 $F'(x)$ 在 $[\theta_1,0]$ 和 $[0,\theta_2]$ 上应用罗尔定理：
    存在 $\eta_1 \in (\theta_1, 0)$ 使得 $F''(\eta_1)=0$。
    存在 $\eta_2 \in (0, \theta_2)$ 使得 $F''(\eta_2)=0$。

    最后对 $F''(x)$ 在 $[\eta_1, \eta_2]$ 上应用罗尔定理：
    存在 $\xi \in (\eta_1, \eta_2) \subset (-1,1)$ 使得 $F'''(\xi)=0$。
    
    而 $F'''(x) = f'''(x) - P'''(x) = f'''(x) - 3$。
    所以 $F'''(\xi) = f'''(\xi)-3=0$，即 $f'''(\xi)=3$。证毕。
